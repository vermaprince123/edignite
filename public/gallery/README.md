# Gallery Images

## How to Add Images

1. **Download images** from Edignite's social media:
   - Instagram: https://www.instagram.com/ed_ignite/
   - LinkedIn: https://in.linkedin.com/company/edignite-ngo
   - Facebook: https://www.facebook.com/socialindiagroup.in/

2. **Name your images** according to the categories:
   - `education-1.jpg`, `education-2.jpg`, etc.
   - `arts-1.jpg`, `arts-2.jpg`, etc.
   - `events-1.jpg`, `events-2.jpg`, etc.
   - `sports-1.jpg`, `sports-2.jpg`, etc.
   - `community-1.jpg`, `community-2.jpg`, etc.

3. **Image Specifications**:
   - Format: JPG or PNG
   - Recommended size: 1200x1200px (square) or 1200x800px (landscape)
   - File size: Optimize to under 500KB for web performance

4. **Update the data** in `lib/data.ts`:
   - Update the `image` field in `galleryImages` array to match your file names
   - Example: `image: "/gallery/education-1.jpg"`

## Current Placeholder System

The gallery currently uses beautiful gradient placeholders with category-specific icons. Once you add real images, they will automatically replace the placeholders.

## Image Categories

- **Education**: Learning sessions, classroom activities, teaching moments
- **Arts**: Artwork, drawing sessions, creative workshops
- **Events**: Foundation day, celebrations, special occasions
- **Sports**: Sports activities, physical games, outdoor activities
- **Community**: Community outreach, volunteer meets, gatherings
